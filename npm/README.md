# ztnet-cli (npm)

This package installs the prebuilt **ZTNet CLI** (`ztnet`) for your platform.

## Install

```bash
npm install -g ztnet-cli
ztnet --help
```

## How it works

On install, `ztnet-cli` downloads the matching `ztnet` binary from the GitHub Releases for the same version.

Supported platforms:

- Windows (x64)
- Linux (x64)
- macOS (arm64, x64)
